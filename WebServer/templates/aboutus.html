About us

In biomedical field data is a great treasure. Using data, in fact we can know more things about biological systems to increase healthcare efficiency. The biggest problem is that data not always is in a standard form, so sometimes is impossibile share data. The main aim of this master thesis project is to create a repository to share sleeping data trying to use all biomedical standards(ihe, hl7). The main actors in this work are: Dauin from Politecnico di Torino, Astel Electronic Engineering and CNR. I am the student that is currently working in this project, Samuele Tozzi.
